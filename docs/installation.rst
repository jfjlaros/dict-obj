Installation
============

The software is distributed via PyPI_, it can be installed with ``pip``:

::

    pip install dict-obj


From source
-----------

The source is hosted on GitHub_, to install the latest development version, use
the following commands.

::

    git clone https://github.com/jfjlaros/dict-obj.git
    cd dict-obj
    pip install .


.. _PyPI: https://pypi.org/project/dict-obj
.. _GitHub: https://github.com/jfjlaros/dict-obj.git
